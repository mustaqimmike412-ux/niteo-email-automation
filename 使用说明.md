# 邮件自动化系统使用说明

## 系统概述

本系统实现了自动化的客户开发信发送功能，包括：
- 客户数据库存储（公司名、网址、邮箱等）
- 网站自动分析（提取行业类型、产品关键词）
- 智能开发信生成（根据行业和邮箱类型定制内容）
- 自动邮件发送（支持公共邮箱和个人邮箱）
- 定时任务执行（每日自动运行）

## 系统架构

```
email_automation/
├── database/

│   └── email_automation.db # SQLite数据库文件

└── main.py               # 主程序入口
```

## 快速开始

### 1. 配置邮箱

编辑 `config/smtp_config.json` 文件：

```json
{
    "smtp_server": "smtp.gmail.com",
    "smtp_port": 587,
    "username": "your_email@gmail.com",
    "password": "your_app_password",
    "sender_name": "Your Name",
    "sender_email": "your_email@gmail.com",
    "use_tls": true,
    "daily_limit": 50,
    "send_interval": 30
}
```

**常见邮箱SMTP设置：**

| 邮箱服务商 | SMTP服务器 | 端口 | 说明 |
|-----------|-----------|------|------|
| Gmail | smtp.gmail.com | 587 | 需要开启两步验证，使用应用专用密码 |
| QQ邮箱 | smtp.qq.com | 587 | 需要开启SMTP服务，使用授权码 |
| 163邮箱 | smtp.163.com | 25/587 | 需要开启SMTP服务，使用授权码 |
| Outlook | smtp.office365.com | 587 | 使用微软账号密码 |
| 企业邮箱 | 咨询管理员 | 25/587 | 根据企业邮箱配置 |

### 2. 系统命令

#### 初始化系统
```bash
python main.py init
```

#### 导入Excel数据
```bash
python main.py import "path/to/your/excel_file.xlsx"
```

#### 发送邮件（手动执行）
```bash
# 发送所有待发送邮件
python main.py send

# 限制发送数量（推荐）
python main.py send --limit 50

# 模拟运行（不实际发送）
python main.py send --dry-run
```

#### 查看统计信息
```bash
python main.py stats
```

## 邮件发送规则

1. **防重复发送**：同一邮箱7天内不会重复发送
2. **邮箱类型识别**：
   - 公共邮箱（info@, support@, sales@等）→ 发送通用开发信
   - 个人邮箱 → 发送个性化开发信，包含联系人姓名
   - 领英邮箱 → 标记为linkedin类型
3. **行业定制**：根据网站内容自动识别行业类型，生成针对性内容
4. **发送限制**：每日最多发送50封，避免被标记为垃圾邮件

## 定时任务

系统已配置每日上午9:00自动执行邮件发送任务。

### 管理定时任务

- **查看任务状态**：在TRAE的自动化任务管理中查看
- **暂停任务**：可以暂停定时任务
- **修改时间**：可以调整执行时间
- **手动触发**：可以立即执行一次

## 注意事项

1. **邮箱安全**：
   - 使用应用专用密码，不要直接使用邮箱登录密码
   - 定期更换密码
   - 建议使用企业邮箱发送开发信

2. **发送频率**：
   - 建议每日发送不超过50封
   - 发送间隔建议30秒以上
   - 避免短时间内大量发送

3. **内容优化**：
   - 定期更新邮件模板
   - 根据回复情况调整内容
   - A/B测试不同主题行

4. **数据备份**：
   - 定期备份 `database/email_automation.db` 文件
   - 保留原始Excel文件

## 故障排除

### 邮件发送失败
1. 检查SMTP配置是否正确
2. 确认邮箱开启了SMTP功能
3. 检查网络连接
4. 查看数据库中的错误日志

### 网站分析失败
1. 检查网站是否可以正常访问
2. 部分网站可能有反爬虫机制
3. 可以手动补充公司信息

### 数据导入问题
1. 确保Excel文件格式正确
2. 检查必要的列是否存在（客户、官网、客户邮箱）
3. 邮箱格式需要正确

## 技术支持

如有问题，请检查：
1. 日志文件（logs目录）
2. 数据库记录（email_logs表）
3. 配置文件（config/smtp_config.json）
